# Postman Newman API Automation

This repository contains an API automation portfolio project built using **Postman** and **Newman**.  
The project is designed to demonstrate API test automation, local execution, HTML reporting, and future CI integration with Jenkins.

## Tools Used

- Postman
- Newman
- Node.js
- Jenkins
- GitHub
